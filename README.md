# DOCARCH - Documentation Architecture Framework

**Meta-framework for consistent, AI-optimized software documentation.**

## Overview

DOCARCH defines a complete documentation architecture for modern projects, especially those involving AI agents. It standardizes how documentation is written, navigated, and maintained.

- **What**: README + SPEC
- **How**: ARCH + Code Docs
- **Why**: ADR
- **Change**: Commits

## Key Features

- Comprehensive DOCSPEC standards
- Quantitative Metrics (Cost α, Utility β)
- Hierarchical & Self-referential design
- AI-Agent ready
